# News-article-clustering-using-K-Means-clustering
OVERVIEW
This project focuses on clustering news articles into different groups using the K-Means clustering algorithm. The goal is to automatically organize similar articles without using labeled data.

OBJECTIVE
To apply K-Means clustering on text data
To group similar news articles together
To understand how NLP and clustering work together

DATASET
The dataset used is the 20 Newsgroups dataset, which contains around 18,000 news articles across multiple topics.
It is directly loaded using Scikit-learn.

TECHNOLOGIES USED
Python
Google Colab
Scikit-learn
NumPy
Matplotlib

METHODOLOGY
1.Load the dataset
2.Perform text preprocessing
3.Convert text into numerical form using TF-IDF
4.Apply K-Means clustering
5.Analyze and visualize clusters
*The code for this project has been attached

RESULTS
1.Articles were grouped into meaningful clusters
2.Each cluster represents a similar topic
3.Demonstrates unsupervised learning effectively

ADVANTAGES
No labeled data required
Easy to implement
Works well for large datasets

LIMITATIONS
Choosing number of clusters can be tricky
Results may vary
Not always perfectly accurate

CONCLUSION
This project shows how K-Means clustering can be applied to text data to group news articles. It is a useful technique for organizing and analyzing large datasets.

HOW TO RUN
1.Open Google Colab
2.Copy the code into a new notebook
3.Run all cells step by step
